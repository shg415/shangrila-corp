# シャングリラ コーポレートサイト デザインリファレンス

参考: 株式会社ACES (https://acesinc.co.jp/)

---

## レイアウト原則

- **全画面セクション**: 各セクションはビューポート高さに近い大きさで設計
- **左寄せコンテンツ**: テキストは左側に配置し、右側にビジュアル要素
- **対角線要素**: ページに動きをもたらす斜めの幾何学的デザイン要素を使用
- **大胆な余白**: コンテンツ周囲に広い余白を設け、息苦しさを排除
- **情報の明確な階層**: ラベル → 大見出し → 本文 の三層構造を徹底

## カラーパレット

```
背景:       #ffffff (純白)
テキスト:    #0d0d0d (ほぼ黒)
サブテキスト: #666666
ボーダー:    #e8e8e8
コーポレート緑: #2d7a4f
緑（薄）:   #e8f5ee
暗い緑:     #1a4d32
幾何学要素:  #1a1a1a (深いチャコール)
```

## タイポグラフィ

- **見出しフォント**: Noto Sans JP (太め: 700, 500) — **セリフフォントは使用しない**
- **本文フォント**: Noto Sans JP (細め: 300, 400)
- **英字表記**: Inter (数字・英語に使用)
- **サイズスケール**:
  - Hero大見出し: clamp(3.5rem, 7vw, 8rem)
  - セクション見出し: clamp(2rem, 3.5vw, 3rem)
  - 本文: 14-15px
  - ラベル: 11px (大文字 + 広いトラッキング)
- **日本語フォントウェイト**: 太字を大胆に使う (700)

## ナビゲーション

- 白背景 + 細いボーダーライン
- ロゴ左端、ナビリンク中央〜右寄り
- CTAボタンは右端に配置（コーポレートカラーで塗りつぶし）
- フォントサイズ小さめ (12-13px)、アイテム間隔は広く

## ヒーロー

- **背景**: 純白または明るい灰色 (#f8f8f8)
- **左半分**: 大きなテキストコンテンツ（ページの60%程度）
- **右半分**: 大胆な幾何学的ビジュアル or グラフィック要素
  - ACESは対角線の暗いブロック要素
  - シャングリラは緑と黒の幾何学的グラフィック（SVGで実装）
- **テキスト**: 漆黒のサンセリフで大きく、イタリック不使用
- **サブコピー**: グレーで小さく、シンプルに

## 幾何学的デザイン要素

- ACES参考: 右側に大きな対角スラッシュ (+45°傾いた矩形ブロック群)
- シャングリラ版: 緑系のジオメトリック要素（四角・線・ドット）
- CSS `clip-path` や CSS transformで実装
- アニメーション: GSAPでゆっくり回転・移動

## セクション構造

1. **Hero** — 純白BG / 大見出し左 / 幾何学ビジュアル右
2. **Ticker/Marquee** — 緑背景 / 事業名が流れる
3. **事業内容 (Services)** — 白BG / リスト形式 or カード（整然としたグリッド）
4. **コンセプト** — チャコール背景 / 白テキスト / インパクト重視
5. **会社概要 (About)** — 白/薄灰BG / 2カラムグリッド
6. **お問い合わせ** — 白BG / シンプルな2カード

## アニメーション (GSAP)

- セクション入場: フェードイン + 縦スライド（ACESより少し遅め: 0.8-1.0s）
- 幾何学要素: ゆっくりと浮遊 or 回転（3-4s loop）
- 数値: カウントアップ
- カーソルホバー: 矢印アイコンが斜め移動

## 避けるべき要素（グラフィックデザイナーサイトとの差別化）

- セリフフォント（Cormorant Garamond, Noto Serif JP）不使用
- 和紙トーンの温かい背景色 → 純白・クールグレーに
- 写真背景のフルスクリーンヒーロー → SVGグラフィックに
- 余白のみの「間（ま）」スタイル → 技術的・構造的なレイアウトに
- ゴールド/アンバー系アクセント → コーポレートグリーンのみ
