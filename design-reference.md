# シャングリラ デザインリファレンス — ACES (https://acesinc.co.jp/)

## ACESの核心的な視覚要素（必ず再現する）

### 1. ヒーロー構造
- 背景: ほぼ白 (#f5f5f5 〜 #ffffff)
- 左下: 大きな太字テキスト（画面の40%程度の高さに配置）
- 右側: 大きな**斜め暗色ジオメトリック要素**（画面右30-40%を占める）
  - 暗い背景の矩形を skewX(-8〜-12deg) または clip-pathで斜めにカット
  - 内部に小さな正方形ブロックが並ぶ（グレー系・透明度あり）
- 左下にタブ切替（Vision / Mission ← 重要）

### 2. ナビゲーション
- ロゴ左端、ナビアイテム多め（7〜8項目）、右端にCTAボタン
- 背景白、下線なし（透明）
- フォントサイズ小さめ（12-13px）

### 3. セクション構造
- 各セクション: 1つの大きなステートメント + サポートテキスト
- 左上に小さなセクション名（/ACESについて のような形式）
- 下部に関連コンテンツ

### 4. サービスセクション
- タブナビ（事業内容 / わたしたちの技術 etc.）
- 大きな1文のキャッチコピー
- 下に2カラムのシンプルなカード（タイトル + 短い説明）

### 5. タイポグラフィ
- フォント: **太めのゴシック** (700-900weight)
- セリフフォント: 不使用
- 大見出し: clamp(2.5rem, 5vw, 5rem)
- セクションステートメント: clamp(1.6rem, 2.5vw, 2.5rem)
- ラベル/タグ: 11-12px, letter-spacing 広め

### 6. カラー
- 背景: #f8f8f8 (わずかにグレーがかった白)
- テキスト: #111 (ほぼ黒)
- ジオメトリック要素: #1a1a1a (深いチャコール)
- アクセント: シャングリラは #2a7a4b (緑)
- グレー: #888, #ccc

## 避けること（グラフィックデザイナーサイトとの差別化）
- セリフフォント絶対不使用
- 和紙トーン背景不使用（純白〜薄グレー）
- 横線ボーダーだらけのリスト形式 → カード形式に
- 過剰な余白 → 適切な情報密度
- 「section-label + 大見出し + 説明文」の3層パターン → 1つのステートメントに集約
